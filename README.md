# ProyectoFinal
Sistema Web para taqueria

---
### Introduccion:
En este proyecto se creo una pagina web en la cual haya un menu digital.

---
###Roles:
- Admin:Puede agregar tacos nuevos y precios, tambien eliminar estos mismos, puede eliminar empleados y ver los usuarios registrados, tambien puede agregar compras a clienes frecuentes.
- Empleado:Puede agregar compras a clientes frecuentes.
- Cliente: Solo puede ver el menu y su perfil.
Todos los roles pueden ver el menu, sus perfiles y descargar el menu.
